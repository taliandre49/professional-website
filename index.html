GUI Maze

Graphical User Interface Game (using Java)

This Java-based GUI project involves guiding McDiver through a sewer system to find a ring and reach the exit within a set number of steps, collecting coins for points. Initially using a depth-first walk, ensuring the ring's discovery but potentially limiting the bonus multiplier, the project emphasizes McDiver's successful exit within the step limit. Scoring involves the product of collected coin values and a seek phase multiplier. A secondary implementation prioritizes the shortest path to guarantee success, followed by optimizing scores through strategic coin collection. Overall, the project underscores McDiver's successful exit within the specified steps.

  • Category: Object Oriented Programming

Scrip-Zip

Electron Desktop App (using Typescrip and JavaScript)

ScripZip, a transformative Electron app developed using TypeScript, streamlines the estimation and migration process for ScriptRunner instances from Jira DC/Server to Jira Cloud. Originally conceived as a desktop application, ScripZip has recently evolved into a versatile Jira plugin. This tool, now a suite of scripts, efficiently automates the retrieval and meta-information extraction of both inline and saved scripts within a ScriptRunner for Jira Server/Data Center Instance. The app achieves this by creating a meticulously structured archive of all scripts, organized by script type (e.g., Behaviours, Scripted Fields, Jobs) in a parent directory named "all-scripts." Additionally, ScripZip generates a detailed diagnostic CSV report for each script, resembling the output of Microscope for workflows. This report includes essential attributes such as the script's last run timestamp, line count, and associated projects/issue types, contributing to a comprehensive solution for managing ScriptRunner instances with precision and ease.

  • Client: Company-based Project created as tool for automation for future clients
  • Role: Associate Software Engineer
  • Category: Front-End, Back-End, Desktop Application, and Jira Plugin Development
